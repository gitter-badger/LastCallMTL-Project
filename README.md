# LAST CALL MTL


## Getting Started

You'll need bower to install the dependencies.
1. Install Bower
2. Run the command below
```sh
bower install
```
3. Open index.html

[How to use Bower, Youtube](https://www.youtube.com/watch?v=Vs2wduoN9Ws)
